Программа "Desktop Image"
автор: Константин Ильяшенко
e-mail: kx13@ya.ru

Назначение:
    Программа предназначена для вывода изображений в формате JPG/GIF/BMP на рабочий стол. Для формата GIF поддерживается прозрачность.
    С помощью этой программы можно, например, отображать "линеечки" на рабочем столе.

Использование:
    Во время первого запуска программа попросить ввести url или полный путь до картинки, которую вы хотите показывать. Если путь не найдет
программа повторит запрос. После того, как введен правильный путь, вы увидете свою картинку на рабочем столе.
    По правой кнопке мыши можно вызвать контекстное меню, с помощью которого можно ввести другой путь к картинке или выйти из программы.

Реестр:
    Программа использует для хранения настроек ветвь реестра: HKEY_CURRENT_USER\Software\desktopimage.

Недостатки:
    Не поддерживается перезагрузка по времени, т.е. изображение меняется лишь при запуске программы или если вы введете новый путь.
    Занимает порядка 6Мб оперативной памяти, это связано с тем, что программа использует СОМ объект IPicture. В будующем возможна
работа над уменьшением занимаемой памяти.

Технические детали:
    Программа была собрана с помощью компилятора lcc-win32 [http://www.cs.virginia.edu/~lcc-win32], но, в принципе, должна собираться
любым компилятором.

Исходный код:
    Можете скачать по желанию. Распостраняется по лицензии BSD.
